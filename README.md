# OWOCNI CRM + Twenty (GitHub package)

Ten folder jest gotowym pakietem do wrzucenia na GitHub.  
Zawiera połączone katalogi:

- `owocni-crm/` — SSOT i runbook migracji
- `twenty/` — analizy, snapshoty POC, artefakty robocze

## Start here

1. `owocni-crm/README.md` — główny routing dokumentacji SSOT.
2. `owocni-crm/CHECKLIST_REVIEW_SZEF.html` — interaktywny review checklist (single source of truth).
